# 胖胖奖励管理

当前版本：`v1.0.0`

一个手机优先的家庭积分与奖励管理应用，用于记录胖胖的每日任务、周任务、周复盘、积分收支和兑换计划。

## 版本说明

`v1.0.0` 是第一版可部署版本，目标是让家庭奖励规则真正跑起来：父母可以每天结算、每周复盘、确认兑换，孩子可以看到积分、任务和愿望进度。应用已支持本地预览存储和 PostgreSQL 正式部署两种模式。

## 功能说明

- 爸爸、妈妈账号登录，权限一致，写操作记录操作者。
- 首页展示剩余积分、本周收入/支出、今日状态、月历、积分流水和今日随机任务。
- 日期页支持每日分档结算、日常动态、父母备注、结算备注、临时任务、周任务和周六复盘。
- 兑换页支持零花钱兑换、父母预设礼品、胖胖许愿计划和确认扣分。
- 任务页维护每日任务、周任务、临时任务模板，并按类型分组展示。
- 设置页维护预算、积分规则、兑换比例、每日保底/上限、兑换模板、随机任务池和视觉皮肤。
- 积分流水支持按时间、收支类型和关键词筛选，并可查看详情和删除修正。
- 每日任务支持未完成、1/3、2/3、完成四档计分，重复结算会覆盖当天每日积分。
- 随机任务每天可抽取一次，完成确认后发放固定积分。
- 视觉采用蛋仔乐园风格，内置开心、悲伤、期待、偷看、兴奋等表情素材。
- 数据使用 PostgreSQL 持久化，部署使用 Docker Compose。

## 本地开发

```bash
npm install
copy .env.example .env
docker compose up -d postgres
npm run prisma:deploy
npm run seed
npm run dev
```

打开 `http://localhost:3000`。

## 生产部署

生产环境推荐使用 `docker-compose.prod.yml`：

```bash
docker compose -f docker-compose.prod.yml up -d --build
```
